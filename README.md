## Аналіз складності алгоритмів



### Рекурсивний алгоритм (FibonacciRecursive)

- Часова складність: O(2ⁿ)
  - Кожен виклик породжує два нових рекурсивних виклики
  - Відбувається повторне обчислення однакових значень

- Просторова складність: O(n)
  - Через глибину стеку рекурсивних викликів



### Ітераційний алгоритм (FibonacciIterative)

- Часова складність: O(n)
  - Використовується один цикл від 2 до n

- Просторова складність: O(1)
  - Використовується фіксована кількість змінних



### Алгоритм динамічного програмування (FibonacciDP)

- Часова складність: O(n)
  - Кожне число Фібоначчі обчислюється один раз

- Просторова складність: O(n)
  - Використовується масив розміру n + 1
